# 💎 Metal-Flex
**High-precision commodity tracker built for the aesthetic web.**

### 🛠 Tech Stack
- **Backend:** Go (Golang) with `shopspring/decimal` for rock-solid financial math.
- **Frontend:** Svelte 5 (Runes) — reactive, lightweight, and modern.
- **Infrastructure:** Monorepo managed by Turbo, deployed on Vercel.
